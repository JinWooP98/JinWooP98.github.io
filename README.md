## 프로젝트 소개
- '랜덤값(Math.random())'을 활용하여 만들 수 있는 간단한 게임 2가지를 만들었습니다.
- 게임은 '같은 그림 찾기'와 '풍선 터트리기'가 있습니다.
- 짧은 시간 내에 클리어를 하도록 유도함으로써 순발력을 기를 수 있습니다.
